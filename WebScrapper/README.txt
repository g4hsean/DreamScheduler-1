Required Libraries:
requests
beautifulsoup4

To install the required libraries first in stall pip then run the following:

1) pip install requests
2) pip install beautifulsoup4

if the above does not work then navigate to the following folder location in a command prompt
python2.7->scripts and execute pip.exe then the install command like above.

to run the script (in command prompt):
python scrapper.py

you can also run it from Python IDLE by opening the file in IDLE then pressing F5 key.
